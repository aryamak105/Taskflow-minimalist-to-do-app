# 📋 TaskFlow – Minimalist To-Do List App

## 📖 Project Overview
TaskFlow is a minimalist and user-friendly To-Do List App UI/UX design created in Figma. The project focuses on improving productivity through a clean interface, intuitive navigation, and an organized task management experience.

## 🎯 Objective
The objective of this project is to design a simple, modern, and responsive task management application that helps users create, manage, complete, and organize their daily tasks efficiently.

## ✨ Features
- ➕ Add New Tasks
- ✅ Mark Tasks as Completed
- 🗑️ Delete Tasks
- 📱 Clean & Minimalist UI
- 🎨 Modern Design
- 👤 User-Friendly Experience

## 🛠️ Tools Used
- Figma
- UI/UX Design Principles

## 📂 Project Structure
- Splash Screen
- Home Screen
- Add Task Screen
- Completed Tasks Screen
